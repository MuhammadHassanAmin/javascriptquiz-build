Promises & then/catch

What Promise pattern does this code implement?
function timeout(promise, time) {
  return Promise.race([
    promise,
    new Promise((_, reject) =>
      setTimeout(() => reject(new Error('Timeout')), time)
    )
  ]);
}

  • This implements a timeout pattern for Promises: 1) Races between the actual operation and a timeout Promise, 2) Automatically rejects if the operation takes too long, 3) Prevents operations from hanging indefinitely, 4) Common pattern for network requests or long-running operations, 5) Helps implement robust error handling for time-sensitive operations, 6) Can be reused across different Promise-based operations.
